# ReplyFlow — AI Email Reply Assistant

Stop staring at emails. Paste any email, pick your goal and tone, get 3 ready-to-send replies in seconds.

## Features
- 5 reply goals: reply, decline, follow up, push back, buy time
- 4 tone options: professional, warm, direct, firm
- Optional context field for more accurate replies
- Multilingual — replies match the language of the email
- Previous replies saved so you can compare
- No login, no signup, free to use

## Tech
Built with vanilla HTML, CSS, and JavaScript. Powered by the Claude API.
